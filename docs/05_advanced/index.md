# Part 5 — Advanced topics

This part connects LLMs to **products**: retrieval (**RAG**), **agents** and tools, **long context**, **multimodal** stacks, **evaluation**, and **safety / hallucinations**.

## Goals

- Design a **RAG** pipeline with explicit failure modes (retrieval quality, grounding).
- Compare **agent** patterns (ReAct, tool calling) to batch inference.
- Use standard **benchmarks** (MMLU, HumanEval, etc.) without overfitting to leaderboard scores.
