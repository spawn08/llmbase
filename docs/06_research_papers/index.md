## How to read a paper entry

1. **One-minute summary** — problem and result.
2. **Architecture / method figure** — SVG in `docs/assets/diagrams/`.
3. **Code** — minimal reproduction or API usage (imports included).
4. **Interview takeaways** — what you must be able to say out loud.
